# Cerebras AI Integration

This document describes how Cerebras AI is integrated and used in the coding agent codebase.

## Overview

Cerebras AI is one of the supported LLM providers in the coding agent. It provides fast inference for large language models through their specialized hardware platform.

## Configuration

### Environment Variables

Cerebras AI is configured through the following environment variables in `.env`:

```bash
# LLM Provider Selection
LLM_PROVIDER=cerebras

# Model Selection (depends on Cerebras available models)
LLM_MODEL=llama-3.3-70b  # Example model name

# API Configuration
CEREBRAS_API_KEY=your-cerebras-api-key
```

### Configuration File

The Cerebras configuration is defined in `src/config.py`:

```python
# Cerebras AI (lines 24-26)
CEREBRAS_API_KEY = os.getenv("CEREBRAS_API_KEY", "")
CEREBRAS_BASE_URL = "https://api.cerebras.ai/v1"
```

## Provider Mapping

Cerebras is mapped to use the OpenAI-compatible API interface:

```python
PROVIDER_MAP = {
    "cerebras": "openai",  # OpenAI-compatible API
}
```

This means the coding agent uses the standard OpenAI client interface to communicate with Cerebras AI.

## Model Initialization

When `LLM_PROVIDER` is set to `cerebras`, the model is initialized with:

```python
def get_model_string() -> str:
    # Returns: "openai:{LLM_MODEL}"

def get_model_kwargs() -> dict:
    kwargs = {}
    if LLM_PROVIDER == "cerebras":
        kwargs["base_url"] = CEREBRAS_BASE_URL
        if CEREBRAS_API_KEY:
            kwargs["api_key"] = CEREBRAS_API_KEY
    return kwargs
```

The actual model initialization happens in `src/agent.py`:

```python
from langchain.chat_models import init_chat_model

model = init_chat_model(
    get_model_string(),  # "openai:{model_name}"
    **get_model_kwargs()  # base_url and api_key
)
```

## Troubleshooting

### Common Issues

1. **API Key Errors**
   - Verify `CEREBRAS_API_KEY` is set in `.env`
   - Check that the API key is valid and active
   - Ensure your Cerebras account has available credits

2. **Connection Errors**
   - Check network connectivity
   - Verify `CEREBRAS_BASE_URL` is correct
   - Check if Cerebras API is operational (status page)

3. **Model Not Found**
   - Verify the model name in `LLM_MODEL`
   - Check Cerebras documentation for available models
   - Model names may change as new versions are released

### Getting an API Key

1. Visit [Cerebras Cloud](https://cloud.cerebras.ai/)
2. Create an account or sign in
3. Navigate to API Keys section
4. Generate a new API key
5. Add it to your `.env` file
